# 기억 장면 재구성 방향

하루결의 핵심 이미지는 단순히 원본 사진을 예쁘게 필터링한 결과가 아니라, 사용자가 찍은 1인칭 사진 조각들을 바탕으로 하루의 대표 장면을 3인칭 다이어리 컷으로 재구성하는 것이다.

## 문제의식

사용자가 하루 동안 찍는 사진은 대부분 1인칭 시점이다.

- 음식 사진
- 한강 풍경
- 꽃 사진
- 카페 컵
- 하늘, 길, 산책 장면

이 사진들은 사용자가 그 장소에 있었다는 증거이지만, 다이어리 한 페이지에 들어갈 대표 이미지는 종종 다른 형태를 원한다.

예를 들어 사용자가 친구와 한강에서 피크닉을 했다면, 실제 사진은 음식, 돗자리, 강, 꽃, 하늘 조각일 수 있다. 하지만 다이어리의 대표 이미지는 `한강을 배경으로 친구와 피크닉을 즐기는 나의 모습`에 더 가깝다.

즉, 하루결은 사진을 그대로 복사하는 앱이 아니라 `내가 본 장면`을 바탕으로 `내가 있었던 하루`를 그려주는 앱을 지향한다.

## 제품 컨셉

### 1인칭 기록에서 3인칭 기억으로

- 입력: 사용자가 찍은 사진 여러 장, 일정, 위치, 기분, 메모
- 분석: 음식/풍경/장소/시간/사람 관계 힌트 추출
- 출력: 사용자가 그 하루 속에 있었음을 느낄 수 있는 대표 장면

예시:

- 입력 사진: 김밥, 한강, 돗자리, 꽃, 노을
- 입력 일정: 친구와 한강 피크닉
- 출력 이미지: 한강 노을 아래 돗자리에서 친구와 음식을 나누는 두 사람의 감성 일러스트

## 얼굴과 개인정보 원칙

정확한 얼굴 복원은 기본 기능으로 두지 않는다. 사용자가 원본 얼굴 사진을 명시적으로 제공하지 않은 이상, 앱은 사용자의 실제 얼굴을 추정해서 만들지 않는다.

대신 다음 표현을 기본값으로 둔다.

- 뒷모습
- 옆모습
- 실루엣
- 손, 어깨, 머리 모양 정도의 약한 개인화
- 사용자가 직접 고른 아바타/헤어/옷 스타일

이 방식은 `내가 있었다`는 감각은 살리면서, 민감한 얼굴 정보는 덜 다룬다.

## 이미지 모드

### 감성화 모드

원본 사진을 유지하면서 색감, 질감, 붓터치, 파스텔 톤을 적용한다.

- 저사양 기기 기본값
- 가장 빠르고 안정적
- 개인정보 위험 낮음

### 기억 장면 모드

여러 장의 사진과 일정 정보를 바탕으로 대표 장면을 재구성한다.

- 하루결의 핵심 차별화 기능
- 사용자는 `내 하루를 누가 멀리서 예쁘게 그려준 느낌`을 받는다.
- 기본 인물 표현은 뒷모습/실루엣/아바타 기반

### 프라이버시 모드

인물의 얼굴과 신체 특징을 더 추상화한다.

- 얼굴 디테일 제거
- 마네킹/캐릭터/실루엣에 가까운 표현
- 공개용 다이어리 이미지에 적합

## 구현 단계

### 1단계: 로컬 파스텔 변환

현재 MVP 단계. 원본 사진을 기기 안에서 파스텔 다이어리 이미지로 변환한다.

### 2단계: 로컬 아트 필터 강화

붓터치, 유화, 수채화, 색연필 질감 등 전통적인 스타일 변환을 추가한다.

- Van Gogh식 붓터치
- 수채화 번짐
- 파스텔 종이 질감
- 경계선 단순화

### 3단계: 템플릿 기반 기억 장면

완전한 diffusion 없이도 시작할 수 있는 방식이다.

- 배경 후보: 사용자가 찍은 풍경 사진
- 전경 요소: 음식/꽃/돗자리/소품 사진에서 추출 또는 아이콘화
- 인물: 사용자 설정 아바타, 뒷모습 실루엣
- 최종: 레이어 합성 후 파스텔/붓터치 필터 적용

이 단계는 로컬에서 가볍게 구현할 수 있고, 저사양 기기에서도 동작 가능하다.

### 4단계: 온디바이스 조건부 생성

고사양 기기에서만 활성화한다.

- Android: MediaPipe Image Generator + condition image/plugin 검토
- iOS: Core ML Stable Diffusion + img2img/ControlNet 계열 검토
- 입력 사진에서 edge/depth/layout을 만들고, 그 구조를 유지한 채 다이어리풍 장면 생성

## 권장 UX

사용자가 결과 이미지를 만들 때 다음 중 하나를 선택하게 한다.

- 원본 감성화: 내가 찍은 사진을 예쁘게
- 기억 장면: 내가 있었던 하루를 한 장면으로
- 프라이버시 장면: 얼굴을 숨긴 공개용 이미지

기본값은 `기억 장면`이 아니라 `원본 감성화`로 둔다. 다만 앱의 대표 기능과 브랜딩은 `기억 장면`을 중심으로 잡는다.

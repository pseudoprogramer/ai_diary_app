# 캘린더-사진 맥락 매칭 에이전트

하루결의 기본 인터페이스는 사용자가 새 캘린더 앱에 적응하게 만드는 것이 아니라, 이미 쓰고 있는 캘린더와 사진첩을 바탕으로 하루의 맥락을 조용히 복원하는 것이다.

## 핵심 흐름

1. 사용자는 기존 Apple/Google/Galaxy 캘린더를 그대로 사용한다.
2. 하루결은 사용자가 허용한 캘린더 목록과 일정 정보를 읽는다.
3. 하루결은 사진첩에서 오늘 촬영한 사진의 시간/장소 정보를 읽는다.
4. AI는 일정의 시간/장소/캘린더 분류와 사진의 시간/장소/장면을 비교한다.
5. `오늘 어떤 일을 했는지`를 추론한다.
6. 하루가 끝날 때 감성 일기와 대표 기억 장면을 만든다.

## 기본 인터페이스

하루결의 홈 화면은 긴 입력 폼보다 `오늘의 맥락 검토`에 가깝다.

### 오늘의 흐름

타임라인 형태로 다음 정보를 보여준다.

- 캘린더 일정
- 해당 시간대의 사진 묶음
- 위치 이동 힌트
- AI가 추론한 활동 라벨

예시:

- 12:00 친구와 점심 / 음식 사진 3장 / 홍대 근처 / `점심 약속`
- 15:00 카페 작업 / 커피 사진 2장 / 합정 카페 / `카페에서 작업`
- 19:00 한강 산책 / 노을, 강 사진 5장 / 여의도 한강공원 / `한강 산책`

사용자는 필요할 때만 수정한다.

- 포함/제외
- 활동 라벨 수정
- 한 줄 메모 추가
- 사진 대표 컷 선택

## 캘린더 연동 정책

### 캘린더 단위 선택

기존 캘린더 분류를 유지한다.

- 집
- 직장
- 학원
- 개인
- 친구
- 여행

각 캘린더는 다음 상태 중 하나를 가진다.

- 항상 포함
- 항상 제외
- 매일 물어보기

AI는 캘린더 이름과 색상도 맥락으로 사용한다.

- `직장` 캘린더: 업무/회의/출근 맥락
- `학원` 캘린더: 수업/공부 맥락
- `집` 캘린더: 생활/휴식/가족 맥락
- `친구` 캘린더: 만남/여가 맥락

### 일정 단위 조정

오늘 생성 전에 일정별 포함/제외를 바꿀 수 있다.

- 민감한 일정 제외
- 오늘 실제로 하지 않은 일정 제외
- 캘린더에는 없지만 사진으로 추론된 활동 추가

## 사진 매칭 정책

사진은 다음 신호를 가진다.

- 촬영 시간
- GPS 위치(EXIF 또는 사진 라이브러리 메타데이터)
- 장면 분류
- 대표 객체: 음식, 카페, 풍경, 꽃, 야경, 사람, 문서 등

일정과 사진은 다음 기준으로 매칭한다.

1. 시간 겹침
2. 일정 전후 여유 시간
3. 위치 근접성
4. 캘린더 분류
5. 사진 장면 분류

예시:

- 일정: `친구와 한강 피크닉`, 13:00~16:00
- 사진: 13:20 김밥, 14:05 돗자리, 15:30 한강 노을
- 추론: `친구와 한강에서 피크닉을 즐김`

## 추론 결과 모델

앱 내부에서는 하루를 다음 구조로 정리한다.

```json
{
  "date": "2026-04-30",
  "segments": [
    {
      "start": "12:00",
      "end": "14:00",
      "sourceCalendar": "친구",
      "eventTitle": "한강 피크닉",
      "matchedPhotos": ["photo_1", "photo_2"],
      "placeHint": "여의도 한강공원",
      "activityLabel": "친구와 한강 피크닉",
      "confidence": 0.86,
      "included": true,
      "userMemo": "바람이 좋았다"
    }
  ]
}
```

이 구조를 바탕으로 일기와 대표 이미지를 만든다.

## AI 생성 입력

일기 생성 모델에는 원본 사진 전체를 보내지 않는 것을 원칙으로 한다. 텍스트 신호 중심으로 보낸다.

- 일정 제목
- 캘린더 이름
- 시간대
- 장소 힌트
- 사진 장면 라벨
- 사용자 메모
- 포함된 세그먼트 요약

이미지 생성/변환은 로컬에서 처리한다.

## 불확실성 UX

AI가 확신하지 못하면 단정하지 않는다.

예:

- `15시쯤 카페 사진이 있었어요. 이 시간을 카페에서 보낸 걸로 기록할까요?`
- `캘린더에는 없지만 한강 사진이 여러 장 있어요. 산책으로 추가할까요?`
- `직장 캘린더 일정은 기본 제외 상태예요. 오늘 회의도 제외할까요?`

## MVP 우선순위

1. 캘린더 목록 가져오기
2. 포함할 캘린더 선택 저장
3. 오늘 일정 가져오기
4. 오늘 사진 가져오기
5. 시간 기준으로 일정-사진 매칭
6. 사용자에게 오늘의 흐름 타임라인 표시
7. 감성 일기 생성
8. 로컬 파스텔/붓터치 이미지 생성

## 장기 방향

하루결은 사용자가 매번 일기를 직접 쓰는 앱이 아니라, 일상의 신호를 모아 `오늘의 이야기를 제안하는 앱`이다. 사용자는 AI가 만든 초안을 확인하고, 민감한 내용은 빼고, 중요한 기억은 한 줄 더하는 방식으로 기록한다.
